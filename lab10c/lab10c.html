The Box Model in CSS

The Box Model in CSS describes how a block elements in HTML are wrapped by a set of imaginary boxes. For example, a DIV element has the following parts associated with it:

  • The Content of the div
  • Some Padding (top,right, bottom, left)
  • A Borders (top,right, bottom, left)
  • A Margin (top,right, bottom, left)

The challenging part of understanding the BOX model may be due to the fact that these imaginary boxes also interact with each other as well as the browser dimensions.

Let's see an examples:

    In the examples below, we have three imaginary boxes around each DIV.
  • Yellow holds the DIV Content and its Padding,
  • Red is the Border, and
  • the Gray area (before and after the HR tag) represent the Margins.

Content of DIV 1:

    DIV with default div properties (no CSS rules)

Content of DIV 2:

  • padding:50px 10px 50px 10px;
    • /* notice the yellow padding starts with Top and bottom of 50, compared to left and right which are 10 (top, right, bottom, left) */
  • border-width: 10px 20px 40px 80px;
    • /* notice the red border starts with top=10, and doubles as we rotate around. (top, right, bottom, left) */
  • border-style: solid;
  • border-color: Red;
  • margin: 40px 20px 10px 20px;
    • /* notice the gray margins left and right are 20, and top is 4 times bigger than bottom.(top, right, bottom, left) */

Content of DIV 3:

  • padding:50px 50px 50px 50px;

  • border-width: 10px 10px 10px 10px;
